                  $items = field_get_items('node', $node, 'field_integer');

                  $node->content['field_integer'] =

Field Type

  • hook_field_schema()
    • Define la estructura de la BD
  • hook_field_info()
    • Define la estructura del campo
  • hook_field_is_empty()
    • Define si nuestro campo está vacío o no
  • Otros hooks...
    • hook_1, Hook_2, ...

Widget

  • hook_field_widget_info()
    • Define la estructura del widget
  • hook_field_widget_form()
    • Define el formulario del widget
  • Otros hooks...
    • hook_field_formatter_form_alter(), hook_field_formatter_WIDGET_TYPE_alter()

Formatter

  • hook_field_formatter_info()
    • Define la estructura del widget
  • hook_field_formatter_settings_form()
    • Define el formulario de configuración del formatter
  • hook_field_formatter_settings_summary()
    • Define la presentación de la configuración el formatter
  • hook_field_formatter_prepare_view()
    • Realiza la lógica necesaria para preparar la presentación del campo.
  • hook_field_formatter_view()
    • Devuelve el renderable del campo
